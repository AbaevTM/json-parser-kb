sc-агент предварительной обработки агентных scp-программ осуществляет проверку корректности связки спецификации sc-события, инициирующего агентную scp-программу*, соответствующей некоторой агентной scp-программе, и, в случае ее корректности, добавляет ее во множество сформированных спецификаций sc-событий. Таким образом, Данный sc-агент активируется при появлении в sc-памяти агентной scp-программы, принадлежащей классу корректных scp-программ.
