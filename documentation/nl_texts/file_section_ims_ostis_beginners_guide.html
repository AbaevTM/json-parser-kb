  2. Познакомиться с основными пожеланиями Технологии OSTIS, используя, в частности, документ Файл. Пояснение. Технология OSTIS
  3. Познакомиться с принципами построения Метасистемы IMS.OSTIS, используя, в частности, документ Файл. Пояснение. Метасистема IMS.OSTIS
  4. Просмотреть структуру (оглавление) Документации Технологии OSTIS и изучить разделы, посвященные моделям, средствам, и методам проектирования интеллектуальных систем

  6. Если Вы хотите принять участие в совершенствовании Технологии OSTIS, т.е. отите войти в состав исполнителей Проекта IMS.OSTIS, смотрите Файл. Раздел. Рекомендации начинающим разработчикам Метасистемы IMS.OSTIS
