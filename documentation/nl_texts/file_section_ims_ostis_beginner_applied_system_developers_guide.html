Для того, чтобы быстро войти в курс дела по использованию Метасистемы IMS.OSTIS для проектирования (разработки) различных интеллектуальных систем, рекомендуется выполнить следующее:
  1. Определить основной набор предметных областей, используемых в базе знаний разрабатываемой интеллектуальной системы
  2. Изучить Библиотеку многократно используемых компонентов баз знаний и проверить, есть ли в ней те компоненты, которые соответствуют предметным областям, используемым в базе знаний разрабатываемой интеллектуальной системы. Такими компонентами прежде всего являются различного вида онтологии указанных предметных областей. Определить набор таких компонентов, которые должны войти в состав базы знаний проектируемой интеллектуальной системы.
  3. Изучить Библиотеку sc-агентов (многократно используемых атомарных и неатомарных sc-агентов) и выбрать те sc-агенты, которые нужны для разрабатываемой интеллектуальной системы
  4. Изучить Библиотеку многократно используемых компонентов пользовательских интерфейсов интеллектуальных систем и отобрать те компоненты, которые нужны для разрабатываемой системы.
  5. Изучить Библиотеку типовых подсистем интеллектуальных систем, направленных на поддержку консультационно-обучающей деятельности, на поддержку совершенствования системы, на поддержку управления процессом совершенствования, на управление информационной безопасностью. Отобрать те типовые подсистемы, которые востребованы для разрабатываемой системы.
  6. Изучить Библиотеку интерпретаторов sc-моделей интеллектуальных систем и выбрать тот интерпретатор, который наиболее подходит для разрабатываемой интеллектуальной системы
  7. Собрать стартовый прототип проектируемой интеллектуальной системы из выбранных многократно используемых компонентов.
  8. Разработать демонстрационно-тестовый диалог конечного пользователя с разрабатываемой интеллектуальной системой и дополнить разрабатываемый прототип интеллектуальной системы такими знаниями и навыками, чтобы указанный диалог мог состояться. Для этого необходимо расширить стартовую версию прототипа интеллектуальной системы новыми специально разрабатываемыми фрагментами базы знаний и новыми специально разработанными sc-агентами, которые желательно предложить в состав соответствующих библиотек многократно используемых компонентов интеллектуальных систем.
  9. Создать проект, направленный на совершенствование полученного прототипа разрабатываемой интеллектуальной системы. Построить иерархическую систему подпроектов и проектных задач для созданного проекта и описать все это в базе знаний проектируемой интеллектуальной системы.
  10. Разработать четкий, достаточно детальный план совершенствования полученного прототипа разрабатываемой интеллектуальной системы и приступить к его выполнению.
