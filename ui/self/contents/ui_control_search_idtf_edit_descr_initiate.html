Поиск sc-элементов по идентификатору осуществляется следующим образом:
  1. сначала необходимо активировать элемент управления поиска sc-элементов по идентификатору, для этого необходимо выполнить щелчек левой клавишей мыши на нем (в нем появится курсор)
  2. с помощью клавиатуры необходимо набрать искомый идентификатор или подстроку, которая в нем содержится. После того, как будет набрано три и более символа, система начнет отображать список с найденными вариантами
  3. если в найденных вариантах присутствует необходимый иденфтикатор, то с помощью щелчка левой клавиши мыши по нему, можно перейти к семантической окрестности данного sc-элемента
